# make-magazine — 슬라이드 덱 매거진 제작 (일반화)

기존 구현( `articles/korean-indie-new-wave.html`, `articles/arise.html` 등 )을 템플릿 삼아
새 주제의 매거진 기사를 만드는 절차. 구조는 고정, **테마(색·폰트·연출)는 주제에 맞게 가변**.

## 입력
- 주제와 기사 제목 (스포티파이 플리 연동 시 플리 제목 = 기사 제목, /playlist-sync 규칙)
- 아이템 목록: 앨범/곡/책/영화/장소 등 6~24개 (제목·작자·연도)
- 필자 페르소나(기본: 비바체 — 결론 선행, "정확히 말하면/구분하자면" 정밀화, 유명세보다 고유한 결, 사족 금지)

## 1. 테마 결정 (주제 → 토큰)
기존 파일의 `:root` CSS 변수만 바꾸면 전체가 변한다. 주제의 무드에서 도출:

| 토큰 | 역할 | 예시 |
|---|---|---|
| `--bg` `--surface` | 배경/카드 | 다크 네이비 `#0D0F16`, 세피아 `#171310`, 페이퍼 `#F4EFE6`(라이트) |
| `--ink` `--muted` `--faint` | 글자 3단계 | 배경과 대비 유지 |
| `--accent` `--accent2` | 포인트 | 음악=크림슨, 재즈=브라스, SF=네온시안, 문학=딥그린, 시티팝=선셋퍼플 |
| `--gold` | 보조 포인트 | 액센트의 보색 계열 |
| `--serif` + @import | 제목 세리프 | 국문 명조(Nanum Myeongjo) / Playfair(라틴) / 고딕만 쓰면 테크 무드 |
| 파티클 색 (JS `cols`) | 배경 연출 | 액센트 RGB로 교체 |

연출 가변 요소: 바이닐 디스크(`.discwrap`)는 음악 전용 — 책이면 책등/종이 질감, 영화면 필름 스프로킷 등으로
교체하거나 제거. 파티클 밀도·글로우(`.slide::before/::after`)도 무드에 맞게 조절. 과장 금지 — 절제가 기본값.

## 2. 구조 (고정)
1. **커버**: kicker(시리즈·호수) → h1 제목(3부작 등 시리즈면 Ⅰ·Ⅱ·Ⅲ 로마자 포함) → 리드 1단락(기사의 관점 선언) → 아트 그리드 → sub(범위·개수·글 비바체)
2. **아이템 슬라이드 × N**: 좌 아트(+연출 오브젝트) / 우 에디토리얼 — `NN / N` 인덱스, 대문자 모노 작자명, 세리프 대제목, 액센트 룰, 모노 메타라인(연도 · 출처 · 장르), 본문 3~5문장(배경→설명→감상)
3. **들으러/보러 가기**: 외부 링크 버튼(플리·서점 등) + 전체 목록. 16개 초과 시 `.listen.dense`
4. **햄버거 내비**: `<!-- ISSUE NAV -->` 블록을 복사해 새 호를 목록에 추가 — **모든 기사 파일의 목록을 함께 갱신**할 것

## 3. 에셋 파이프라인
- 커버 아트: iTunes Search API(`itunes.apple.com/search`, 600x600 치환) → 실패 시 Deezer(`api.deezer.com/search`, cover_xl). 리마스터/디럭스 표기는 메타라인에서 제거
- 저장: `articles/covers/<slug>/NN.jpg`
- **몽타주 검증 필수**: PIL로 격자 이미지 만들어 눈으로 전수 대조 (오매칭은 반드시 나온다)

## 4. 검증 (필수)
- 로컬 서버 + Playwright: 데스크톱 1600×900과 **모바일 360×780** 둘 다 스크린샷
- 모바일은 `@media(max-width:768px){:root{--su:1.05vh}}` 스케일 재정의가 있는지 확인 — 없으면 세로 화면에서 전부 작아진다
- 목록형 슬라이드가 화면을 넘치는지 확인

## 5. 배포
- 루트 `index.html` 기사 목록에 카드 추가 (4-썸네일 그리드 + 제목 + 모노 메타)
- 이 레포 커밋 규칙: 단일 커밋 히스토리 — `git commit --amend -m "GROOVE — 비바체의 음악 노트"` 후 작업 브랜치와 main **둘 다 force push**. 커밋 명의는 로컬 config(sckahn), Claude 트레일러 금지
- 스포티파이 연동이 필요하면 /playlist-sync 스킬 참조

## 금지
- 사족(불필요 배지·그라데이션·과한 글로우), 인터뷰 인용, 작명 자랑, 물성 수사
- 아이템을 "트랙"으로 잘못 지칭 (앨범이면 앨범)
